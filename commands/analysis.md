# Code Analysis Menu

This project has several code analysis commands available. Please select which analysis you'd like to perform:

## Available Commands

1. **analyze_code** - Perform deep code analysis
   - Understand unfamiliar code
   - Detect potential issues
   - Summarize complex logic

2. **knowledge_graph** - Create a contextual knowledge graph
   - Visualize relationships between modules, classes, and functions
   - Map the codebase architecture

3. **optimize** - Optimize and improve code quality
   - Performance improvements
   - Refactoring suggestions
   - Best-practice updates

4. **quality** - Evaluate code quality
   - Readability analysis
   - Structure assessment
   - Consistency checks
   - Potential bug detection

## Your Selection

Which analysis would you like to perform? You can specify:
- `analyze_code` for deep code analysis
- `knowledge_graph` for architecture visualization
- `optimize` for performance and maintainability improvements
- `quality` for overall quality assessment

If you provided an argument ($ARGUMENTS), I'll execute that analysis. Otherwise, please let me know which one you'd like to use.
